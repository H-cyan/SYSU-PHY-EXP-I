1.1. Latex¶

Pearson相关系数计算公式： $$ r = \frac{(\sum_{i=1}^{n}{X_i-\bar{X}})({Y_i-\bar{Y}})}{\sqrt{\sum_{i=1}^{n}{(X_i-\bar{X})^2}}\sqrt{\sum_{i=1}^{n}{(Y_i-\bar{Y})^2}}} $$

def FFT(y, t):
    fs =500
    # signal
    N = len(t)
    # fft
    yf = fft(y)
    xf = fftfreq(N, 1/fs)[:N//2]
    yf_nor = 2.0/N * np.abs(yf[0:N//2]) # normalization
    xf_half = xf[:len(yf_nor)]
    return xf_half,yf_nor
